# Algorithm Performance Analysis

## Overview
This repository analyzes and compares:
- Randomized Quicksort
- Deterministic Quicksort
- Hash Table with Chaining

The project combines theoretical analysis and empirical evaluation
to study efficiency and scalability.

## Files
- quicksort.py: Quicksort implementations
- hashtable.py: Hash table using chaining
- benchmark.py: Performance comparison

## How to Run

python benchmark.py
